Spotify Angular Assement

by don Pablo

Artist Search:

Spotify Angular Test, In this test, you will write a music search app using the Spotify api, ngxs state management library and Angular 9+. All api calls must be initiated through the store. All data to be stored within the store. Navigation must be done via the store. Data to be displayed on the UI, must be retrieved from the store using Selectors. The Artists page, Albums Details Page, must have buttons to go back to the previous page. Api calls must be kept too as few as possible, using the store as a cache. The app must be responsive. The general styling of the app is left up to the developer and should style it as best as possible.
